Localized Messages

All of the CV Sieve stylesheets have been localized. Simply stated, this means you can easily change the output of CV Sieve to a different language. To do so, go to the xsl/common/messages directory and make a copy of the en.xml file, giving it a name that represents your language. For example, if you want to create a French localization, you should use the two-letter language code for French, fr. You can then edit this new fr.xml file and translate all the English messages to their French equivalents.[4]

Once you have created a localized message file, you can tell CV Sieve to use it simply by setting the XSL parameter lang to the appropriate language code. Continuing with the French example above, if you are using the included Ant helper script, you would change the résumé example to French by adding the following string to the cvsieve.params variable of the build.properties file:

lang=fr


[4] If you have created a localization file for your language, please consider submitting it to the CV Sieve maintainer so that it can be included in the distribution for others to use.
