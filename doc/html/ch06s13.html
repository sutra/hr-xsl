Achievements

This element provides a place to list honors and awards.

Examples

<Achievements>
    <Achievement>
        <Date>
            <StringDate>2001-2002</StringDate>
        </Date>
        <Description>Graduate Fellowship</Description>
        <IssuingAuthority>Intel Foundation</IssuingAuthority>
    </Achievement>
</Achievements>

CV Sieve Output

Achievements

     * Graduate Fellowship. Intel Foundation, 2001-2002

Notes

The HR-XML website has more documentation.
