Quick Start

  1. Locate the directory where you unpacked CV Sieve and go to the examples/resume sub-directory

  2. Open the file resume.xml and, using it as a template, edit as needed

  3. On the command line, go to the examples/resume directory and type ant html. This will create an HTML version of the résumé. (For a PDF or plain text version, type ant pdf or ant text, respectively.)
